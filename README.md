# ApiSamples

#### 介绍
通过我们提供的接口，你可以轻松获取物联网设备信息和采集到的数据，还可操作终端设备的启停，再配合您的程序可实现高效的物联网数据采集与控制。在此之前， 您应该拥有一个ECIOT V3.0系统分配的管理账号，并且建立了能够正常采集数据的设备与链路。
